# Acc102-track4-baijiu
# Baijiu Stock Analysis Tool (2022–2026)
## 1. Product Overview
This is an interactive financial analysis tool built with Streamlit. It provides visual analysis and comparison for major Chinese baijiu stocks, using pre-downloaded historical data and the CSI 300 Index as a market benchmark.

## 2. Analytical Problem
Many finance students and beginner investors lack a simple, visual tool to analyse long-term stock performance, risk, and return without coding.

## 3. Target User
- Finance and economics students
- Individual investors
- Users analysing the Chinese baijiu industry

## 4. Dataset
- Period: 2022‑01‑01 to 2026‑04‑21
- 9 listed baijiu companies
- Benchmark: CSI 300 Index
- Data source: Pre-downloaded historical financial data

## 5. Python Methods Used
- Pandas: Data processing and indicator calculation
- Matplotlib: Data visualization
- Streamlit: Interactive web interface
- NumPy: Numerical calculation

## 6. Key Features
- Single stock analysis
- Dual stock comparison
- 6 professional financial charts
- Automatic performance metrics
- Professional analysis and suggestions

## 7. Product Link
https://acc102-track4-baijiu-hy9jgerekkem26.streamlit.app/

## 8. How to Run
1. Install dependencies:
pip install -r requirements.txt

2. Run the application:
streamlit run app.py

## 9. Limitations
- Uses pre-downloaded historical data, not real-time data
- Focuses on technical indicators only

## 10. Future Improvements
- Connect to real-time financial data API
